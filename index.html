Bienvenue

Valais Festival est un projet valaisan, qui a pour but de répertorier toutes les associations et tous les spectacles qui ont lieu durant une période donnée, afin de mettre en avant l'incroyable diversité culturelle de notre canton.

Stack

Valais Festival est un projet qui utilise les techos suivantes : Python3, Django1.7, Postgres et PostGIS, JQuery, JS, HTML5 et CSS3. Il est prévu pour tourner sur un server Linux, avec Gunicorn et Nginx en reverse proxy
